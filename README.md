# VoiceRobot

VoiceRobot 是一个类似于豆包的语音交互智能体 APP，致力于打造便捷、高效且具趣味性的智能语音交互体验。项目分为 4 个主要模块，结构清晰，易于维护和扩展。

## 项目结构

```
VoiceRobot/
├── app/           # 移动端应用及用户界面 (UI)
├── voiceengine/   # 语音 SDK 封装与接口
├── core/          # 基础功能与框架
├── lottie/        # 动效资源与通用UI组件
```

- **app/**：APP主程序，包括用户界面和相关业务逻辑。
- **voiceengine/**：第三方语音SDK的封装层，对外提供统一的语音服务接口。
- **core/**：核心基础功能，包括通用工具、数据结构和底层逻辑。
- **lottie/**：lottie动效动画文件和各类可复用的资源组件。

## 功能简介

- 离线/在线语音识别与合成
- 智能语音交互与对话管理
- 拓展个性化语音包
- 丰富的界面动效
- 支持多平台适配
